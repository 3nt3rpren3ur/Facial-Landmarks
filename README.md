### Notes
* Landmark points
   * The mouth can be accessed through points [48, 68].
   * The right eyebrow through points [17, 22].
   * The left eyebrow through points [22, 27].
   * The right eye using [36, 42].
   * The left eye using [42, 48].
   * The nose using [27, 35].
   * The jaw using [0, 17].
